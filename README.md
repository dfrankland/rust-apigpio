# Raspberry PI GPIO access library based on pigpiod.

This library a pure-Rust analogue to the C `pigpio_if2` library,
providing an async interface to talk to pigpiod.

For documentation, see [docs.rs](https://docs.rs/apigpio/).

For source code, issues, etc.,
see [Salsa](https://salsa.debian.org/iwj/rust-apigpio).
